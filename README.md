## Pokedex 😁

Creación de una pokedex utilizando la api (https://pokeapi.co/). En este sitio web se puede visualizar todos los pokemon con su imagen, pasar de pagina y hacer busquedas.

### Tecnologías 💻

* HTML
* CSS
* Bootstrap 5
* JavaScript
